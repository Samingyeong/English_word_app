# 영단어 암기 앱 📚

엑셀 파일을 업로드하여 영단어를 학습할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 📤 **엑셀 파일 업로드**: 엑셀 파일을 업로드하면 자동으로 단어를 분리하여 Day별 단어장으로 저장
- 📅 **Day별 단어장 관리**: Day1, Day2, Day3... 형식으로 귀여운 UI로 단어장 표시
- 🗑️ **단어장 삭제**: 불필요한 단어장을 쉽게 삭제할 수 있습니다
- 🎴 **플래시카드 모드**: 카드를 뒤집어가며 단어를 외우는 모드
  - 학습 방향 선택: 영어→한글, 한글→영어, 랜덤
  - 카드를 여러 번 뒤집어 확인 가능
- ⌨️ **직접 입력 모드**: 단어를 보고 뜻을 입력하는 모드
  - 학습 방향 선택: 영어→한글, 한글→영어, 랜덤
- 🔀 **랜덤 학습**: 단어 순서를 랜덤으로 섞어서 학습
- ❌ **오답 관리**: 틀린 단어를 자동으로 모아서 추가 공부 가능
- 💾 **로컬 스토리지 저장**: 브라우저에 데이터가 자동으로 저장되어 다음에도 사용 가능

## 시작하기

### 1. 패키지 설치

npm을 사용하는 경우:
```bash
npm install
```

또는 pnpm을 사용하는 경우:
```bash
pnpm install
```

### 2. 개발 서버 실행

npm을 사용하는 경우:
```bash
npm run dev
```

또는 pnpm을 사용하는 경우:
```bash
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 엑셀 파일 형식

엑셀 파일은 다음과 같은 형식이어야 합니다:

| 영어 | 한글 |
|------|------|
| apple | 사과 |
| banana | 바나나 |
| cat | 고양이 |

- 첫 번째 열: 영어 단어
- 두 번째 열: 한글 뜻
- 첫 번째 행은 헤더로 인식되지 않습니다 (데이터로 처리됩니다)

## 사용 방법

1. **단어장 추가**: 메인 화면에서 엑셀 파일을 업로드하면 자동으로 Day1, Day2... 형식으로 단어장이 생성됩니다.

2. **학습하기**: 
   - Day 카드를 클릭하면 학습 모드를 선택할 수 있습니다.
   - **플래시카드 모드**: 카드를 뒤집어가며 단어를 외웁니다.
   - **직접 입력 모드**: 한글 뜻을 보고 영어 단어를 입력합니다.

3. **오답 관리**: 
   - 틀린 단어는 자동으로 오답 목록에 추가됩니다.
   - 오답 목록에서 "오답 공부하기" 버튼을 클릭하면 오답만 모아서 다시 학습할 수 있습니다.

## 기술 스택

- **Next.js 14**: React 프레임워크
- **TypeScript**: 타입 안정성
- **Tailwind CSS**: 스타일링
- **xlsx**: 엑셀 파일 파싱
- **lucide-react**: 아이콘
